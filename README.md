# CodSec Malware Automation Presentation

A professional, interactive presentation showcasing CodSec's malware detection and response automation solution.

## Features

- **21 Interactive Slides** covering the complete solution:
  - Malware detection workflow
  - False positive management
  - Critical systems protection
  - ROI analysis and case studies
  - Integration with existing tools
  - Implementation roadmap

- **Bootstrap 5.3 Framework** - Modern, mobile-responsive design

- **Professional Pricing Calculator**:
  - Tiered pricing (Basic, Pro, Enterprise)
  - Deployment models (Rental/Purchase)
  - Contract term discounts
  - Real-time ROI calculations
  - Competitor comparison

- **Interactive Elements**:
  - Bottom progress navigation with dots
  - Keyboard navigation (Arrow keys)
  - Live configuration sliders
  - Dynamic metrics
